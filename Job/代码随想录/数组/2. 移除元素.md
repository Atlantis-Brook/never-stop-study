[LeetCode：27. 移除元素](https://leetcode.cn/problems/remove-element/)
- **双指针**
```java
class Solution {
	public int removeElement(int[] nums, int val) {
	
		int slow, fast;
		for (fast = 0, slow = 0; fast < nums.length; fast ++ ) {
		
			if (nums[fast] != val)
				nums[slow ++ ] = nums[fast];
		
		}
		
		return slow;
		
	}
} 
```

- 时间复杂度：O(n)
- 空间复杂度：O(1)
